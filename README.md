# gimp-color-palettes

A collection of color palettes for GIMP and Inkscape.

## Format

```
GIMP Palette
Name: Palette-Name
Columns: 7
# Comment
  0    0    0  Black
255  255  255  White
```

* `Columns: <number>` - is just an indication for display.
* `# <comment>` - comments must start with a `#`.
